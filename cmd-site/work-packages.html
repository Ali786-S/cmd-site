Explore the project timeline and click on a work package to learn more.

Months 6–15

Work Package 1 · Construction of Migration Crisis Discourses

Corpus-driven and critical discourse analysis of institutional glossaries, statistical categories and public narratives that construct the current ‘migration crisis’.

View Work Package 1
Months 16–26

Work Package 2 · Deconstruction of Migration Crisis Discourse

Participatory action research workshops with illegalised migrants to challenge hegemonic narratives and co-create new vocabularies and stories.

View Work Package 2
Months 27–32

Work Package 3 · Reconstruction of Just Migration Discourse

Intercultural competence workshops with journalists, policymakers and educators using the Toolkit for Action to test alternative narratives in institutional contexts.

View Work Package 3
Months 30–36

Work Package 4 · Toolkit Refinement & Training Programme

Refinement of the Toolkit for Action and development of an ongoing training programme, online workshops and learning resources for wider dissemination.

View Work Package 4
